Getting Started

Welcome to OpenWeb Starter!

You can start building your website in under 2 minutes. No coding experience required.

Step 1: Choose a Template

Browse our collection of professional templates and select one that fits your needs:

  • Portfolio: Perfect for designers, developers, and creatives
  • Resume: Professional resume layout for job seekers
  • Landing Page: Great for products, services, or personal branding

Step 2: Customize Your Content

Use our intuitive editor to customize your website:

  1. Click on any element to edit it
  2. Use the code editor for advanced customization
  3. Preview your changes in real-time
  4. Save your progress at any time

Step 3: Deploy Your Website

Once you're happy with your website, deploy it with one click to popular hosting platforms or download the files.

Using the Editor

Editor Interface

The editor features a split-screen interface:

  • Left Panel: Code editor with syntax highlighting
  • Right Panel: Live preview of your website
  • Top Bar: Save, deploy, and download options

Custom Domain Setup

To use a custom domain with your deployed website:

  1. Purchase a domain from your preferred registrar
  2. Configure DNS settings to point to your hosting platform
  3. Add the domain in your hosting platform's settings
  4. SSL certificates are automatically provisioned

Troubleshooting

Common Issues

Editor not loading

If the editor fails to load, try these steps:

  • Clear your browser cache
  • Disable browser extensions
  • Try a different browser
  • Check your internet connection

Preview not updating

If the live preview stops updating:

  • Check for JavaScript errors in the console
  • Refresh the page
  • Try the reset button

Deployment failed

If deployment fails:

  • Check your account permissions
  • Verify repository settings
  • Try downloading and manually uploading

API Reference

Enterprise Feature

API access is available for Enterprise customers. Contact us to learn more.

Authentication

All API requests require authentication using an API key:

curl -H "Authorization: Bearer YOUR_API_KEY" \
     https://api.openwebstarter.com/v1/templates

Endpoints

GET /v1/templates

Retrieve all available templates

POST /v1/projects

Create a new project from a template

PUT /v1/projects/{id}

Update an existing project
